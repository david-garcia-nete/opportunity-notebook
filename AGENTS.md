# Opportunity Notebook

Opportunity Notebook helps users identify, evaluate, pursue, and track income opportunities.

## First User

David Garcia

## Stack

- Laravel 13
- Blade
- Tailwind
- MySQL
- Redis
- Sail
- PHPUnit

## Product Philosophy

The goal is not to apply to more jobs.

The goal is to make better career and income decisions.

## MVP Entities

- Opportunity
- Contact
- Action
- Application
- Project

## Development Rules

- Prefer Laravel conventions.
- Use feature tests.
- Keep controllers thin.
- Use Form Requests when validation grows.
- Avoid premature AI features.
- Avoid unnecessary complexity.
